# Revived IT

Revived IT is an Australian specialist supplier of refurbished rugged laptops and tablets. We provide durable Panasonic Toughbook, Dell Latitude Rugged and Getac devices used by field technicians, mechanics, mining contractors, agriculture, marine services and other harsh-environment industries.

All devices are fully refurbished and tested in-house. We offer custom configuration options including RAM, SSD and WWAN upgrades, as well as accessories, docks and replacement parts.
